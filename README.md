# coc-asynctasks

asynctasks support for coc

## Install

`:CocInstall coc-asynctasks`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-asynctasks-keymap)`

## Lists

`:CocList demo_list`
